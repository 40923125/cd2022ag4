uarm(NX)

心得:

我覺得NX1980這套CAD功能非常的專業化,在繪製時相關設定很多,能做到許多CAD都做不到的功能,正是因為如此,這套軟體在業界極其泛用,但也是因為可自訂的部分極多,導致繪製過成過於繁雜,對於初學者不太友善,但其先進的技術為設計者提供非常好的使用體驗,加上可程式化的介面,大幅降低了所許的繪製時間。

期末考周

1. 請各學員利用 https://github.com/mdecycu/cd2022_guide/tree/main/downloads/w8_midterm/w8_uarm_nx12/uarm_robot_nx12 目錄中的 uArm 機械手臂零件, 修改 "nx12絞鍊"與 "front joint" 零件的外型, 使能將 "Triangular connecting rod", "left link" 與 "end link" 等零組件, 與 "right crank" 與 "right link" 的組件左右互換. 各學員完成新 uArm 機械手臂組合檔案後, 請將零組件檔案放入個人 cd2022 倉儲 downloads 中的 w18/uarm 目錄後執行 acp, 並以 "完成期末考週 uarm 新組立檔案" 作為提交訊息.

2. 請各組員在完成題目 1. 的任務後, 在個人 cd2022 網頁中, 新增一個 H1 格式標題為 "期末考週" 的頁面, 並將題目 1. 的零組件繪圖過程, 以 Wink 影片展示過程細節說明, 並在頁面中說明所採用的 NX 套件版本與題目 1. 的任務執行心得.
3. 請在完成新幾何外型組立的 uArm 機械手臂後, 將組立件轉入 CoppeliaSim 4.3.0 rev3. 後, 套用 https://www.coppeliarobotics.com/helpFiles/en/simUI-widgets.htm 圖形介面, 以 slider 控制前三軸的轉動. 完成後, 請在個人網頁的 "期末考週" 頁面, 加入第二個 Wink 影片展示過程細節說明.
4. 請在完成新幾何外型組立的 CoppeliaSim 場景後, 請與同組組員合作, 透過 https://mde.tw/cd2022_guide/content/Visualization%20Stream.html 模式, 在 Teams 會議中, 以畫面分享及錄影功能, 由兩位 (含) 以上組員測試題目 3. 的 XML UI slider 控制功能. 完成錄影後, 請將影片 upload 至 Youtube 並嵌入參與學員的個人 "期末考週" 網頁後, 再整合至各分組網頁中的 "期末考週" 頁面, 並納入參與測試學員的心得.
5. 請接續題目 4. 的畫面分享與影片錄製模式, 由兩位 (含) 以上組員協同在新幾何外型組立的 uArm 機械手臂場景中, 加入 suctionPad 零件, 以手動操控或程式操控的方式, 錄製 block 方塊零件的取放模擬影片. 完成錄影後, 請將影片 upload 至 Youtube 並嵌入參與學員的個人 "期末考週" 網頁後, 再整合至各分組網頁中的 "期末考週" 頁面, 並納入參與測試學員的心得.

6. 請接續題目 5. 的畫面分享與影片錄製模式, 由兩位 (含) 以上組員協同在新幾何外型組立的 uArm 機械手臂場景中, 以跨網路 Python remote API 的方式, 錄製圖形介面 slider 控制前三軸的轉動影片. 完成錄影後, 請將影片 upload 至 Youtube 並嵌入參與學員的個人 "期末考週" 網頁後, 再整合至各分組網頁中的 "期末考週" 頁面, 並納入參與測試學員的心得.

控制 uarm 吸起方塊

心得:

40923125:

一開始還不知道吸盤要怎麼接上去,後來觀看學長的影片得知可以利用joint定位,而吸盤自帶吸起功能,只要開啟碰撞就可吸起方塊,用模擬的方式來完成實體的模擬雖然不會有成本問題,但很耗時間。

40923126:

延續利用作業3來增加吸盤與block，原本叫出吸盤後無法吸起，經過討論後才發現吸盤碰撞與方塊的碰撞程式沒有開啟。

40923127:看了學長的影片才知道吸盤怎麼叫出來，不燃還在add裡找半天，然後方塊的碰撞檢測要打開才吸得起來。
